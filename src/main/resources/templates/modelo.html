Modelo Matemático ICE-CREAM

Descripción del Modelo

Este modelo matemático fue desarrollado para optimizar la cantidad de helados que deben trasladarse desde el almacén hacia el mostrador principal de la heladería ICE-CREAM. El objetivo principal es maximizar la ganancia esperada teniendo en cuenta:

  • Precio de cada helado.
  • Probabilidad de demanda de cada sabor.
  • Stock disponible en almacén.
  • Capacidad máxima del mostrador.
  • Cantidad mínima que debe exhibirse de cada producto.

Variables de decisión: Las variables de decisión del modelo son X1, X2, X3, ..., X16, donde cada variable representa la cantidad de unidades de cada sabor de helado que deben trasladarse desde el almacén hacia el mostrador.


Objetivo del modelo: El modelo busca maximizar la ganancia esperada total de la heladería ICE-CREAM, teniendo en cuenta los precios, la probabilidad de demanda, la capacidad del mostrador y las restricciones de stock disponibles.

Paso a Paso para Resolver el Modelo en Excel con Solver

  1. Se creó una tabla en Excel con los precios, probabilidades de demanda, stock disponible y variables de decisión.
  2. Se definieron las variables Xi como la cantidad de helados enviados del almacén al mostrador.
  3. Se construyó la función objetivo multiplicando: Precio × Probabilidad × Xi.
  4. Se sumaron todos los resultados individuales para obtener la Función Objetivo (F.O).
  5. Se agregaron las restricciones: capacidad máxima, stock disponible, mínimos de exhibición y no negatividad.
  6. Se habilitó el complemento Solver desde: Archivo → Opciones → Complementos → Solver.
  7. En Solver se configuró:

    Objetivo: Maximizar la F.O.
    Variables: X1 hasta X16.
    Restricciones: R1-R33.
    Método: Simplex LP.
  8. Finalmente Solver encontró la solución óptima respetando todas las restricciones planteadas.
